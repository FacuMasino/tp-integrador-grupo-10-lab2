\doxysubsection{Detailed Description}
Clase que representa una plantilla de correo electrónico. 

Esta clase se utiliza para cargar una plantilla de correo electrónico desde un archivo, reemplazar los parámetros de la plantilla con valores específicos y obtener el contenido HTML resultante.
