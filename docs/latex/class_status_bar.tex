\doxysubsection{Detailed Description}
Clase que representa una barra de estado. 

La clase \doxylink{class_status_bar}{Status\+Bar} se utiliza para mostrar y actualizar el estado de ciertos datos, como la cantidad de vacunas pendientes, la cantidad de aplicaciones pendientes y el nombre del veterinario.
