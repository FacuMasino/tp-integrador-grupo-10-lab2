\doxysubsection{Detailed Description}
Clase que representa una vista de lista. 

La clase \doxylink{class_list_view}{List\+View} se utiliza para mostrar una lista de elementos en una interfaz de usuario. Permite agregar celdas, columnas y un título, y luego mostrar la lista en pantalla.
