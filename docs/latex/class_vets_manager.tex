\doxysubsection{Detailed Description}
Clase que gestiona la carga, edición, visualización y eliminación de veterinarios. 

La clase \doxylink{class_vets_manager}{Vets\+Manager} se encarga de administrar la información de los veterinarios, permitiendo cargar nuevos veterinarios, editar su información, mostrar la lista de veterinarios y eliminar aquellos marcados como dados de baja.
