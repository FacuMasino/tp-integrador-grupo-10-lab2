Detailed Description

Clase que gestiona la carga, edición, visualización y eliminación de veterinarios.

La clase VetsManager se encarga de administrar la información de los veterinarios, permitiendo cargar nuevos veterinarios, editar su información, mostrar la lista de veterinarios y eliminar aquellos marcados como dados de baja.

◆ load()

void VetsManager::load ( )

Método para cargar y guardar información de un veterinario.

Solicita al usuario un ID de veterinario único, verifica su disponibilidad, carga la información del veterinario mediante un formulario y la guarda en un archivo. Si el ID ya existe, permite reintentar o salir. Si no se completa el formulario, se sale.

El método finaliza mostrando mensajes de éxito o error según el resultado de la operación.
