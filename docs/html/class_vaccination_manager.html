Detailed Description

Clase que gestiona las vacunaciones.

La clase VaccinationManager se encarga de administrar las vacunaciones, incluyendo la carga, edición, visualización y notificación de las mismas. También proporciona métodos para realizar búsquedas y verificar la existencia de un ID de vacunación.
