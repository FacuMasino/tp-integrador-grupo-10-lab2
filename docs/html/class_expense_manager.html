void load ()
 Carga y guarda un nuevo gasto después de solicitar y verificar un ID único. Muestra un formulario para ingresar un ID de gasto único, verifica su disponibilidad en el archivo, y luego carga y guarda los detalles del nuevo gasto utilizando otro formulario. Si el usuario cancela el ingreso o hay errores, la función se interrumpe.
 
void edit ()

Muestra la lista de gastos existentes, solicita al usuario el ID de la transacción a modificar y luego permite la edición de los detalles del gasto utilizando un formulario.
