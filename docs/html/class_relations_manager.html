Detailed Description

Clase para gestionar las relaciones entre mascotas y clientes.

La clase RelationsManager se encarga de cargar, editar, mostrar y guardar las relaciones entre mascotas y clientes. Permite crear nuevas relaciones, editar relaciones existentes, buscar relaciones por ID de mascota y cliente, y verificar la existencia de una relación específica.
