Detailed Description

Clase que representa un administrador de ventas.

Esta clase se encarga de administrar las ventas, cargar, editar y mostrar información relacionada con las ventas. También proporciona métodos para buscar ventas por ID y verificar la existencia de un ID en las ventas.
